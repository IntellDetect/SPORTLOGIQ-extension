## 🇷🇺 Русский

**Sportlogiq Tagger Pro** — расширение Chrome для удобной работы на https://app.sportlogiq.com.

### Установка
1. Скачайте этот репозиторий папкой (**Code → Download ZIP**, затем распакуйте) или `git clone`.
2. Откройте `chrome://extensions`.
3. Включите **Режим разработчика** (переключатель справа вверху).
4. Нажмите **Загрузить распакованное расширение** и выберите эту папку.
5. Значок расширения появится на панели Chrome.

### Вход
1. Нажмите на значок расширения → **Login with Telegram**.
2. В открывшейся вкладке Telegram нажмите **Start** и вернитесь в браузер.
3. Доступ только для участников доверенных рабочих чатов; иначе будет отказано.

### Использование
Откройте https://app.sportlogiq.com — инструменты включаются автоматически после входа. Без действующей сессии расширение
не работает. Если сессия истекла, на странице появится подсказка войти снова.

**Sportlogiq Tagger Pro** — розширення Chrome для зручної роботи на https://app.sportlogiq.com.

### Встановлення
1. Завантажте цей репозиторій папкою (**Code → Download ZIP**, потім розпакуйте) або `git clone`.

4. Натисніть **Завантажити розпаковане розширення** та виберіть цю папку.

Відкрийте https://app.sportlogiq.com — інструменти вмикаються автоматично після входу. Без чинної сесії розширення не працює.
Якщо сесія завершилася, на сторінці з’явиться підказка увійти знову.

## 🇬🇧 English

**Sportlogiq Tagger Pro** is a Chrome extension that streamlines work on https://app.sportlogiq.com.

### Install
1. Download this repository as a folder (**Code → Download ZIP**, then unzip) or `git clone`.
2. Open `chrome://extensions`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** and select this folder.
5. The extension icon appears in the Chrome toolbar.

### Sign in
1. Click the extension icon → **Login with Telegram**.
2. In the Telegram tab that opens, tap **Start**, then return to the browser.
3. Access is limited to members of the trusted work chats; otherwise it is denied.

### Use
Open https://app.sportlogiq.com — the tools turn on automatically once you are signed in. The extension does nothing without a
live session; if your session ends, an in-page prompt asks you to sign in again.
